## About me

I'm a data scientist and software engineer based in Nigeria, with 4+ years building ML models and full-stack web products. My core stack is **Python, Django, React, TypeScript, and PostgreSQL**, and I work comfortably across web, mobile, and cloud deployments.

I founded **[Expert Hive Tutors](https://experthivetutors.com/)**, an EdTech platform that connects students with verified tutors — including a React web client, Django REST API, and React Native mobile app.

I'm affiliated with the **University of Lagos** and publish research on **[ResearchGate](https://www.researchgate.net/profile/Tobi-Samuel-2)** covering computer vision, medical imaging, security systems, and control engineering.
