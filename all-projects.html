Real-time Intrusion Detection System on Windows üîê

Completed

This project implements a Real-time Intrusion Detection System (IDS) for Windows, leveraging Machine Learning techniques to detect network intrusions.

  • Supervised Learning: Random Forest model for identifying known attacks (CICIDS 2018 & SCVIC-APT datasets)
  • Unsupervised Learning: Autoencoder model for anomaly detection
  • Platform: Windows OS
  • Web Interface: Accessible via http://localhost:5000

Setup Instructions:

  • Install Python 3.9 from the official Python website (ensure "Add to PATH" is checked)
  • Install Npcap 1.71 from the official Npcap website
  • Create virtual environment: python3.9 -m venv venv
  • Activate it: . /venv/Scripts/activate
  • Install dependencies: python -m pip install -r requirements.txt
  • Run app: python application.py
